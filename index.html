MoMAC

MoMAC is a user-guided web app for learning about martial arts. Using the styles of engagement as a basis, users can vote for the best martial arts or techniques in the various styles. In addition to contributing new martial arts and techniques, users can add photos to the photo gallery to illustrate martial arts and links to YouTube videos to demo various techniques.


BeerTynder

BeerTynder is a mobile app based off the popular dating app, Tinder. Our application allows users to swipe through randomly generated beers from around the world, view detailed information about those beers, and add to personal lists. By tapping the "Social" tab, users can view other users' selected beers and add to their own wishlists. Users can also search from the BreweryDB database of beers to add a beer manually. This is a collaborative project with other Nashville Software School students.

MoTMe

MoTMe (Message or Text Me) is a web application built in ASP.NET and AngularJS. Users can add other users to their contacts list. Users can then message their contacts. As an alternative, users can also send a text message to their contact, simply by checking a box. The project can be viewed on Github.


Star Data

Star Data is a web app for browsing various statistics from Star Wars. Search planets, starships, vehicles, people, films, and species. Then filter the results. Star Data uses the SWAPI API.

Location Information

Location Information is a web app for getting the city when given a zip code, or getting a list of zip codes when given a city. Search zip codes from all around the world. Get the zip codes of any US city.
